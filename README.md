# AMIV website

This is the home of the new AMIV website.

## How to use

```
npm install
npm start &   # start webpack
npm run server &   # start developer server
```

Lint:
```npm run lint```

## Developer 

Backend: [AMIV API](https://github.com/amiv-eth/amivapi)

Technologies:
 * [Mitrhil](https://mithril.js.org/)
